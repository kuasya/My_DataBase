# My_DataBase
Big Data Infrastructure. Project focused on building a SQL database -- Java Interface

_______________________________________
## Основные этапы реализации:

Лабораторная работа №1: Определение интерфейсов и проектирование базовой структуры.
Включает разработку спецификаций интерфейсов (API) и моделирование основной структуры объектов (entities)

Лабораторная работа №2: Локальная реализация интерфейсов.
Полностью реализуется кодовая база, позволяющая взаимодействовать с локальной версией базы данных средствами Java

Лабораторная работа №3: Дистанционное размещение базы данных.Инфраструктура переносится на удалённый сервер, обеспечивая внешний доступ через протоколы REST или gRPC

Лабораторная работа №4: Организация резервного копирования и репликации.
Настраиваются механизмы автоматического резервного копирования и репликации данных для повышения отказоустойчивости и надежности

Лабораторная работа №5: Масштабирование путем шардинга.
Организуется механизм горизонтального масштабирования для равномерного распределения нагрузок и увеличения производительности.
