## Лабораторная работа №1. Определение интерфейсов 

Цель: создать структуру классов и определить API-интерфейсы для взаимодействия с будущей базой данных.
Цель:Создать каркас программы для дальнейшей реализации собственной базы данных. Основная задача заключается в определении интерфейсов и базовых конструкций для будущего API, которое обеспечит удобное взаимодействие с будущими компонентами системы.

Задачи:  

- Реализовать абстрактные интерфейсы и классы для основного функционала (CRUD).
- Предусмотреть возможность расширения функциональности.
- Продемонстрировать понимание принципов проектирования программного обеспечения.

Язык программирования: Java
